# Object classification with a pre-trained model

## How to run

Install the dependencies.

```bash
npm install
```
or

```bash
bun install
```

Run the app with

```bash
npm run watch
```

or 

```bash 
bun run watch 
```

It should open your browser at `http://localhost:1234`
